# JamoMoa Updates

[자모모아](https://github.com/hobbyworker/JamoMoa) — macOS 한글 NFD/NFC
파일명 정규화 유틸리티 — 의 자동 업데이트 채널.

소스 저장소가 아닙니다. 이 저장소는 두 가지 메타데이터만 호스팅합니다:

- `appcast.xml` — 앱 안의 Sparkle이 fetch 하는 업데이트 피드
- `releases/` — 각 버전의 릴리즈 노트 (`JamoMoa-X.Y.Z.md`)

DMG 바이너리는 이 저장소의 [Releases](https://github.com/hobbyworker/JamoMoa-updates/releases)
에 첨부되어 있습니다. 소스 코드는 [공개되어 있지
않습니다](https://github.com/hobbyworker/JamoMoa).

## 수동 다운로드

자동 업데이트가 무언가 문제로 안 되면 [Releases](https://github.com/hobbyworker/JamoMoa-updates/releases)
페이지에서 직접 DMG를 받아 설치할 수 있습니다.

## 라이선스

자모모아 본체는 비공개 소프트웨어입니다. 이 저장소의 메타데이터(appcast,
release notes)는 사실상의 public domain — 자유롭게 인용/링크하셔도 됩니다.

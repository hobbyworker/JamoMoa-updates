# JamoMoa v0.9.10

한국어 시스템에서 앱 표시명이 드디어 "자모모아"로 보입니다 (이전엔
"JamoMoa"로 잘못 표시).

## 변경

- **앱 이름 한국어 표시** — Finder, Dock, Spotlight, About 윈도우 등에서
  *한국어 시스템*은 **자모모아**, 영어 시스템은 *JamoMoa*, 일본어는
  *濁点ピタッ*, 베트남어는 *Nối Dấu*로 적절히 표시됩니다. 변경 사항이
  적용되려면 *기존 .app을 한 번 제거 후 새로 설치*가 권장됩니다 (Sparkle
  자동 업데이트로는 macOS Finder의 이름 캐시가 옛 이름을 잠시 들고 있을
  수 있음 — 그 경우 Finder 재시작 또는 로그아웃/로그인으로 새로고침).
- **저작권 표기 한국어 번역** — 정보 탭의 copyright가 한국어 시스템에선
  *© 2026 취미생활자*로, 다른 언어에선 *© 2026 hobbyworker*로 표시.

## 내부 정비 (사용자 화면 변화 없음)

- 모든 UI 문자열의 *키*를 한국어에서 영어로 일관되게 정리. 코드 검색 /
  Localizable.xcstrings 편집 모두 표준 macOS 인디 앱 패턴으로 정렬됨.
- macOS의 `LSHasLocalizedDisplayName` 키를 명시 — Finder가 .lproj
  번역 lookup을 *수행하도록* 트리거. 이 키 없으면 macOS는 파일시스템
  이름만 표시 (자모모아가 v0.9.9까지 빠졌던 함정).

변환 동작 / 단축키 / 설정 항목 — 전부 v0.9.9와 동일.

# JamoMoa v0.9.7

내부 정비용 release — *자동 업데이트 채널을 자체 도메인으로 이전*.
사용자 화면에 보이는 변경은 없습니다.

## 변경

- 업데이트 채널을 `hobbyworker.github.io` 에서
  `jamomoa-updates.hobbyworker.me` 로 이전. 두 URL이 현재는 같은
  GitHub Pages backend를 가리키지만, 미래에 호스팅 backend를 옮겨야
  할 때 DNS 만 바꿔서 기존 설치본이 그대로 update를 받을 수 있도록
  *URL의 소유권*을 확보해두는 작업입니다.
- 기존 v0.9.6 이하 사용자는 이번 업데이트를 받은 뒤로 자동으로 새
  URL을 사용합니다. 추가 설정 불필요.

변환 동작은 v0.9.6과 동일.

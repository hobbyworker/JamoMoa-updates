# JamoMoa v0.9.6

v0.9.5 이후 쌓인 사용자 경험 개선을 한 번에. 변환 동작 자체는 변동 없습니다.

## 새 기능

- **앱 아이콘** — 자모모아 자체 아이콘이 추가되어 정보 탭 / Dock /
  알림 등에 표시됩니다.
- **UI 언어 선택** — 설정 ▸ 일반 ▸ 언어. 시스템 / 한국어 / English /
  日本語 / Tiếng Việt. 변경은 *다시 실행 후* 반영됩니다.
- **화면 모드** — 설정 ▸ 일반 ▸ 화면 모드. 시스템 / 주간 / 야간을
  *즉시* 반영. 다시 실행 불필요.
- **Dock에 아이콘 표시** — 설정 ▸ 일반. 평소엔 메뉴바만 (accessory),
  켜면 Dock 아이콘 + 앱 메뉴바도 함께. 다시 실행 불필요. 메뉴바
  자모모아는 양쪽 모드에서 항상 보입니다.
- **변환 → 되돌리기** — 변환한 항목 옆에 **되돌리기** 버튼. NFC → 원본
  NFD 이름으로 디스크 상태를 복원합니다. 하단 바의 **전체 되돌리기**
  로 한꺼번에도 가능 (`되돌릴 항목이 있을 때만 노출`).
- **복사 시 확장자 포함** 옵션 — 빠른 옵션 셋에 추가. 파일에만 적용
  (디렉터리는 점이 식별자의 일부인 경우가 많아 항상 통째로 복사).
- **글로벌 단축키** — 어디서든 키 조합으로 메인 창을 띄울 수 있습니다.
  설정 ▸ 일반 ▸ 단축키에서 자유롭게 등록. 등록 안 하면 동작하지 않으며,
  Sandbox 친화적인 방식이라 추가 권한이 필요 없습니다.
- **라이센스 보기** — 설정 ▸ 정보. 자모모아가 사용하는 오픈소스 패키지의
  라이센스 원문을 모달로 보여줍니다.

## 개선

- **자모모아에 관하여** 메뉴 항목 (Dock 켰을 때 보이는 macOS 앱 메뉴
  기준)이 이제 *설정 ▸ 정보* 탭으로 바로 연결됩니다. 이전에는 macOS
  기본의 아이콘+버전만 보이는 미니 윈도우가 떠서 정보 surface가
  두 군데로 갈렸습니다 — 한 곳으로 통일했습니다.
- **항상 맨 위에 표시**가 켜진 상태에서 설정 윈도우도 함께 floating
  하도록 변경. 이전에는 설정이 메인 창에 가려서 클릭으로 끌어올릴 수
  없는 deadlock이 발생할 수 있었습니다.

# JamoMoa v0.9.12

v0.9.11의 개발자 모드 기능을 다듬은 마이너 정정 릴리즈입니다.
v0.9.11에서 추가된 기능(개발자 모드 + 검색·저장·영속화 가능한 로그
창)은 그대로 유지됩니다.

## 변경

- **메뉴바 아이콘 교체** — 이전 *가가* (`textformat` SF 심볼)에서
  자모모아 브랜드 마크 *ㅈ*로 교체. macOS template 이미지 규약에 맞춰
  라이트/다크 모드와 메뉴 active 상태에 따라 자동으로 색이 맞춰집니다.
- **로그 윈도우 제목** — *로그* → *자모모아 로그*. Dock 우클릭 윈도우
  목록과 Windows 메뉴에서 다른 앱의 로그 창과 한눈에 구분됩니다.
- **메뉴바 드롭다운 정리** — 개발자 모드의 *로그 표시* 항목 제거. 로그
  윈도우는 *설정 ▶ 개발자*의 **로그 윈도우 열기** 버튼 한 곳으로
  통일했습니다. 개발자 모드를 안 쓰는 일반 사용자의 메뉴바가 더
  깔끔해집니다.

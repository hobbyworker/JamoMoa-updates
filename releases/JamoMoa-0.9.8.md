# JamoMoa v0.9.8

설치 DMG 외형 polish. 앱 동작은 v0.9.7과 동일합니다.

## 변경

- **설치 DMG layout** — DMG를 열면 *자모모아 아이콘 (좌측)*과
  *Applications 폴더 단축 alias (우측)*가 나란히 표시되어, drag-drop
  으로 곧장 설치할 수 있습니다. 표준 macOS 인디 앱의 install UX와 동일.
- *이 변경은 새로 DMG를 다운로드 받아 설치하는 경우에만 보입니다.*
  기존 v0.9.x 사용자가 Sparkle 자동 업데이트로 받는 경우엔 설치 흐름이
  내부적으로 끝나므로 layout과 무관합니다.

## 알려진 차이 없음

변환 동작 / 단축키 / UI / 설정 항목 — 전부 v0.9.7과 동일.

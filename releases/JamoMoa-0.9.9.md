# JamoMoa v0.9.9

설치 DMG 배경에 자모모아 화살표 그래픽 추가. 앱 동작은 v0.9.8과
동일합니다.

## 변경

- **DMG 배경 이미지** — 새로 DMG를 열면 자모모아 아이콘과 Applications
  폴더 사이에 *오른쪽 방향 화살표*가 그려져, drag-drop 설치 흐름이
  시각적으로 더 명확해집니다.
- *기존 v0.9.x 사용자에게 영향 없음* — 자동 업데이트는 DMG 내용을
  드러내지 않고 진행됩니다. DMG 배경 변경은 새 install 시점에만 보입니다.

변환 동작 / 단축키 / UI / 설정 — 모두 v0.9.8과 동일.

# JamoMoa v0.9.13

이번 업데이트의 핵심은 **양방향 변환**입니다. 지금까지는 분해된(NFD) 이름을
합치는(NFC) 한 방향만 가능했지만, 이제 **합치기 ⇆ 분해**를 어떤 파일에도
자유롭게 적용할 수 있습니다. 그밖에 메인 창의 **빠른 옵션을 직접 고르고
정렬**하는 기능, 설정 **내보내기 / 가져오기 / 초기화**, 그리고 **수만 개
파일도 끊김 없이 처리**하는 성능 개선이 들어갔습니다.

## 새 기능

- **양방향 변환 (합치기 ⇆ 분해)** — 분해된(NFD) 이름은 *합치기*로 NFC가
  되고, 합쳐진(NFC) 이름은 *분해*로 NFD가 됩니다. 앱이 변환한 적 없는
  파일도 자유롭게 분해할 수 있어, 기존의 "변환 후 되돌리기"(이제 *되돌리기*는
  사라지고 *분해*로 일반화)보다 훨씬 유연합니다. 하단에 **전체 합치기 /
  전체 분해**도 추가했습니다.
- **빠른 옵션 커스터마이즈** — 설정 ▶ 빠른 옵션에서 메인 창 하단 빠른 옵션
  바에 **어떤 설정을 보여줄지 직접 고르고 순서를 정할** 수 있습니다. 각
  항목은 평소엔 깔끔하게 한 줄로 보이고, 마우스를 올리면 출처(예: *일반 ›
  앱*)가 살짝 나타납니다. (항상 표시로 고정도 가능)
- **설정 내보내기 / 가져오기 / 초기화** — 설정 ▶ 일반 하단에서 모든 설정을
  `.plist` 파일로 내보내거나 불러오고, 기본값으로 초기화(경고 확인)할 수
  있습니다. 창 위치 같은 시스템 상태는 빼고 실제 설정만 담깁니다.

## 성능

- **대용량 처리** — 목록을 새 방식으로 바꿔 **수만 개 파일도 스크롤이 끊기지
  않습니다**. 변환 알고리즘의 병목을 제거해 전체 합치기 / 분해가 훨씬
  빨라졌고, 처리 중 **진행 개수**(예: `처리 중… (500/50000)`)도 함께
  표시됩니다.

## 변경

- **설정 화면 정리** — 일반 / 기능 / UI / 빠른 옵션 / 개발자 / 정보로 탭을
  나누고, 각 탭 안도 섹션으로 묶었습니다 (언어 · 화면 모드 · Dock 아이콘은
  UI 탭으로 이동).
- **파일명 복사 옵션 묶음** — *자모 합치기* / *확장자 포함* 을 "파일명 복사"
  그룹으로 정리.
- **로그 윈도우** — 저장 · 최신으로 이동 · 지우기 버튼이 *빠른 옵션* 접힘
  여부와 상관없이 항상 보이도록 변경.
